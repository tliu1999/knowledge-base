# 思考库

一个本地可打开的轻量思考库，用来沉淀观点、面试题、自我复盘、党员角色理解、产品经理思考和 AI 时代观察。

## 使用方式

直接用浏览器打开 `index.html`。

## 内容维护

- 固定思考条目在 `data/entries.js` 中维护。
- 页面上的“提交给我审核”会先写入当前浏览器的 `localStorage`，再打开 GitHub Issue 页面提交给维护者审核。
- 如果希望长期保存新增内容，建议把重要条目整理回 `data/entries.js`。

## 当前内置分类

- 党员角色
- 产品思考
- AI 时代
- 大模型概念
- 机器学习基础
- 自我复盘
- 面试题库
- Mac 常用快捷键
- GitHub / Git 常用指令

## 检索方式

支持按标题、分类、标签、正文和追问内容搜索。
