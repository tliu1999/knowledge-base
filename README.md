# 个人知识库

一个本地可打开的轻量知识库，用来记录常问问题、Mac 快捷键、GitHub/Git 常用指令等内容。

## 使用方式

直接用浏览器打开 `index.html`。

## 内容维护

- 固定知识条目在 `data/entries.js` 中维护。
- 页面上的“添加到本机知识库”会写入当前浏览器的 `localStorage`，适合临时补充和快速记录。
- 如果希望长期保存新增内容，建议把重要条目整理回 `data/entries.js`。

## 当前内置分类

- Mac 常用快捷键
- GitHub / Git 常用指令

## 检索方式

支持按标题、分类、标签、正文和命令内容搜索。
